# AI Automation Website

A modern Next.js website featuring AI voice agents and automation services.

## 🚀 Quick Deploy

### Option 1: Vercel (Recommended for Next.js)
1. Push this repo to GitHub
2. Connect to Vercel
3. Deploy automatically

### Option 2: Netlify Drag & Drop
1. Drag the entire project folder to [netlify.com/drop](https://app.netlify.com/drop)
