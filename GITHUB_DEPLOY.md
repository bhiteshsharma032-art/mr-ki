## 📋 Step-by-Step Instructions:

### 1. Create GitHub Repository
1. Go to [github.com](https://github.com)
2. Click "New repository"
3. Name it: `ai-automation-website`
4. Make it public
5. Click "Create repository"

### 2. Upload Your Code
**Option A: GitHub Web Interface**
1. Click "uploading an existing file"
2. Drag ALL files from your `mr kie website` folder
3. Commit with message: "Initial commit"

**Option B: Git Commands** (if you have Git installed)
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/ai-automation-website.git
git push -u origin main
```

### 3. Deploy with Vercel
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Click "New Project"
4. Select your `ai-automation-website` repo
5. Click "Deploy"
6. **Done!** Your site is live in 2 minutes
